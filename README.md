# Ecom API Testing using Postman

## Project Overview

This project demonstrates API testing of an E-commerce REST API using Postman.

The collection covers authentication, user profile retrieval, product listing, and single product retrieval.

---

## Tools

- Postman v11
- DummyJSON REST API
- JavaScript (Postman Tests)
- Environment Variables

---

## APIs Covered

- Login API
- Get User Profile
- Get Products
- Get Single Product

---

## Validations

- Status Code Validation
- Response Time Validation
- Content-Type Validation
- Response Body Validation
- Data Type Validation
- Environment Variable Validation
